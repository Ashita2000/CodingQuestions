# Sort List 

Given the head of a linked list, return the list after sorting it in ascending order.

<b>Example 1:</b> </br>
<b>Input:</b> head = [4,2,1,3]</br>
<b>Output: </b> [1,2,3,4]</br>

<b>Example 2:</b> </br>
<b>Input:</b> head = [-1,5,3,4,0] </br>
<b>Output: </b>  [-1,0,3,4,5] </br>

<b>Example 3:</b> </br>
<b>Input:</b> head = []
<b>Output: </b> []
